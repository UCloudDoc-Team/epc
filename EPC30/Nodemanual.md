# EPC节点组

EPC节点组是以高性价的快杰云主机为底座，提供高性能计算节点服务。支持一次性创建一台或者多台高性能主机，按需创建，弹性扩容。

EPC节点组支持的规格如下：
| CPU（核） | 内存（GB） | 系统盘（GB） |
|-----|-----|-----------|
| 8 | 16 | 200 |
| 32 | 64 | 200 |
| 32 | 128 | 200 |
| 64 | 128 | 200 |
| 64 | 256 | 200 |
| 96 | 192 | 200 |
| 128 | 256 | 200 |
| 240 | 480 | 200 |  
| 32 | 256 | 200 | 
| 64 | 512 | 200 |
| 96 | 768 | 200 |
