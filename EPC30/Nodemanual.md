# EPC节点组

EPC节点组是以高性能的快杰云主机为底座，提供高性能计算节点服务。支持一次性创建一台或者多台高性能主机，提供标准型和大内存型多种规格节点，满足多种业务场景需求。同时支持自定义计算软件以及自有授权软件，例如Gromacs、Lammps等，客户无需配置环境，操作便捷，开机即用。
另外EPC节点组资源创建时，自动挂载共享存储，支持多点挂载，满足多 

EPC节点组支持的规格如下：

| 32 | 64 | 200 |
| 32 | 128 | 200 |
| 64 | 128 | 200 |
| 64 | 256 | 200 |
| 96 | 192 | 200 |
| 128 | 256 | 200 |
| 240 | 480 | 200 |  
| 32 | 256 | 200 |
